# Satellite Orbit Simulator

A 3D visualization of satellites orbiting Earth using Three.js and satellite.js.

## Features
- Real-time satellite position calculation
- Color-coded satellite categories
- Interactive 3D Earth visualization
- Orbital path visualization
- Responsive controls

## Requirements
- Modern web browser with WebGL support

## Usage
1. Open `index.html` in a web browser
2. Use mouse to rotate and zoom the view
3. Toggle orbit paths and labels using the control panel

## Libraries Used
- Three.js - 3D graphics library
- satellite.js - Satellite position calculations
